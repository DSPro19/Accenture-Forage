## 📊 Accenture Job Simulation – Social Buzz Analysis

Completed Accenture’s virtual job simulation focused on analyzing social media content trends for Social Buzz, a global tech brand. The project involved data cleaning, modeling, and extracting insights from high-volume post data.

### 🔍 Key Highlights

* Identified top 5 content categories and peak engagement times.
* Found "Animals" and "Science" to be most positively received.
* Recommended content strategies for brand partnerships (e.g., Healthy Eating with food brands).

### 🛠 Tools Used

* Excel, Data Cleaning, Data Analysis, Insight Generation
